# Projeto Integrador - Correção de Parâmetros Químicos

## Sobre o projeto
Aplicativo desenvolvido em JAVA com o objetivo de auxiliar técnicos e operadores no processo de correção dos parâmetros de graduação alcoólica e concentração de sacarose durante a fabricação de bebidas.

O sistema busca facilitar cálculos e reduzir dificuldades operacionais no controle químico das bebidas.

## Desenvolvimento

##Objetivo Geral 
Desenvolver uma funcionalidade computacional para auxiliar técnicos e operadores no processo de correção dos parâmetros de graduação alcoólica e concentração de sacarose em bebidas, visando melhorar a gestão da qualidade, reduzir erros operacionais e aumentar a eficiência do processo produtivo. 

### Objetivos Específicos 
• Analisar o processo produtivo de bebidas, com foco nas etapas de correção de graduação alcoólica e concentração de sacarose;  
• Identificar as principais dificuldades enfrentadas por técnicos e operadores durante a execução dos cálculos de correção;  
• Levantar os requisitos necessários para o desenvolvimento de uma ferramenta computacional de apoio ao processo;  
• Estudar os conceitos teóricos relacionados à gestão da qualidade, controle de processos e parâmetros químicos de bebidas;  
• Desenvolver uma aplicação ou funcionalidade que automatize os cálculos de correção dos parâmetros de bebidas;  
• Validar a funcionalidade proposta por meio de testes práticos ou simulações no ambiente produtivo;  
• Avaliar os resultados obtidos com a utilização da ferramenta, considerando a redução de erros, tempo de operação e facilidade de uso;  
• Propor melhorias no processo produtivo com base nos resultados obtidos. 

## Tecnologias utilizadas
- JAVA

## Funcionalidades
- Correção de graduação alcoólica
- Correção de concentração de sacarose
- Auxílio operacional para técnicos
