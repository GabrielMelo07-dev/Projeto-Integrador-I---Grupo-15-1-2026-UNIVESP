# Projeto Integrador - Correção de Parâmetros Químicos

## Sobre o projeto
Aplicativo desenvolvido em JAVA com o objetivo de auxiliar técnicos e operadores no processo de correção dos parâmetros de graduação alcoólica e concentração de sacarose durante a fabricação de bebidas.

O sistema busca facilitar cálculos e reduzir dificuldades operacionais no controle químico das bebidas.

## Tecnologias utilizadas
- JAVA

## Funcionalidades
- Correção de graduação alcoólica
- Correção de concentração de sacarose
- Auxílio operacional para técnicos
